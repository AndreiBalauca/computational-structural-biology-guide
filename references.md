# References and Further Reading

## Purpose

This file lists useful public resources for learning computational structural biology, molecular visualisation, docking, molecular dynamics, and AI-predicted structures.

---

## Structure Databases

| Resource | Use |
|---|---|
| RCSB PDB | Experimental protein and complex structures |
| AlphaFold DB | Predicted protein structures |
| UniProt | Protein sequence, function, domains, and variants |
| GPCRdb | GPCR structures, residue numbering, and diagrams |

---

## Literature and Ligand Databases

| Resource | Use |
|---|---|
| PubMed | Scientific literature |
| ChEMBL | Bioactivity and ligand data |
| BindingDB | Protein-ligand binding data |
| DrugBank | Drug and target information |

---

## Molecular Visualisation

| Tool | Use |
|---|---|
| PyMOL | Protein structure viewing and figure making |
| ChimeraX | Structure visualisation and analysis |
| VMD | Molecular dynamics trajectory visualisation |
| NGL Viewer | Browser-based structure viewing |

---

## Docking and Cheminformatics

| Tool | Use |
|---|---|
| AutoDock Vina | Open-source molecular docking |
| GNINA | Docking with machine-learning scoring |
| RDKit | Cheminformatics and molecule handling |
| Open Babel | Molecular file conversion |

## Molecular Dynamics

| Tool | Use |
|---|---|
| AMBER / AmberTools | Molecular dynamics setup, simulation, and analysis |
| GROMACS | Molecular dynamics simulation package |
| CHARMM-GUI | System building, especially membrane systems |
| MDAnalysis | Python-based trajectory analysis |
| MDTraj | Python-based MD trajectory analysis |

## AI Structure Prediction

| Tool | Use |
|---|---|
| AlphaFold | Protein structure prediction |
| AlphaFold Server | Protein and complex prediction |
| Boltz | Biomolecular structure prediction |
| Chai-1 | AI-based biomolecular modelling |
| Protenix | AI-based structure prediction |

---

## Key Takeaway

Good computational work depends on using reliable public databases, reading the relevant literature, checking structures carefully, and documenting the tools used.
